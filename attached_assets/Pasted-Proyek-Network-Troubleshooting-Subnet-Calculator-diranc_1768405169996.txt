Proyek Network Troubleshooting & Subnet Calculator dirancang sebagai aplikasi web responsif tanpa backend menggunakan HTML, CSS, dan JavaScript (Auto Refresh/Vite) yang dapat digunakan dengan nyaman di desktop maupun perangkat mobile, dengan tujuan menunjukkan pemahaman network fundamentals sekaligus pola pikir troubleshooting yang sistematis dan terstruktur. Aplikasi ini terdiri dari beberapa layar utama, yaitu Subnet Calculator untuk menghitung network address, broadcast, host range, subnet mask, dan total host; OSI Layer Guide yang menjelaskan setiap layer OSI beserta gejala umum dan langkah pengecekan; Command Cheatsheet yang memuat perintah jaringan penting seperti ping, tracert/traceroute, dan nslookup untuk berbagai sistem operasi; serta Case Simulation yang menyajikan contoh permasalahan jaringan nyata lengkap dengan gejala, langkah diagnosis, akar masalah, dan solusi. Seluruh interaksi pengguna dilengkapi dengan UI state yang jelas, meliputi idle state dengan instruksi penggunaan, validating state saat input diproses, success state untuk menampilkan hasil perhitungan, serta error state dengan pesan kesalahan spesifik apabila input tidak valid. Logika subnet dirancang anti error dengan validasi ketat pada format IP (empat oktet dengan rentang 0–255) dan CIDR (0–32), di mana input yang tidak valid akan langsung ditolak tanpa menjalankan perhitungan, sehingga mencegah hasil yang keliru atau aplikasi mengalami crash. Seluruh perhitungan dilakukan sepenuhnya di sisi klien dengan penanganan khusus untuk kasus ekstrem seperti CIDR /0, /31, dan /32, disertai penjelasan singkat agar hasil mudah dipahami oleh pengguna non-teknis. Antarmuka aplikasi dirancang mobile-first dengan navigasi yang jelas, feedback visual yang konsisten, tombol bantu seperti reset dan copy hasil, serta tata letak yang tetap rapi di layar kecil. Pengujian dilakukan melalui berbagai skenario IP valid dan tidak valid, CIDR ekstrem, serta pengujian tampilan di perangkat mobile dan desktop, sehingga menghasilkan aplikasi yang stabil, ringan, minim risiko error, dan sangat relevan sebagai bukti kompetensi dasar jaringan dan troubleshooting untuk kebutuhan rekrutmen.
